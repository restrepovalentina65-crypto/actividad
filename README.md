Sistema de Registro de Citas Fisioterapéuticas

Descripción
Este proyecto consiste en el diseño de un formulario web para registrar citas médicas de pacientes en un sistema fisioterapéutico.

El sistema permite almacenar información del paciente, su estado de salud, citas previas, actividades realizadas y generar una fórmula médica automáticamente.

Objetivo
Aplicar HTML y CSS para construir una interfaz funcional basada en un wireframe.

Contenido del repositorio
wireframe.pdf - Diseño del formulario (estructura visual) 
index.html - Interfaz con etiquetas semánticas 
styles.css - Estilos visuales del formulario 
README.md - Información del proyecto 

Tecnologías utilizadas
-HTML
-CSS
